# Slay the Spire (2019) — Mega Crit Games

> **Genre:** Deckbuilder Roguelike | **Perspective:** 2D Card | **Players:** Solo | **Steam Rating:** 97% Positive

## 🎮 Core Concept

Ascend the Spire by building a deck of cards, collecting relics, and making strategic decisions at branching path nodes. Each of 4 characters has a completely unique card pool and playstyle.

| Character    | Archetype | Card Count | Key Mechanics                                                  |

| **Defect**   | Robot     | 75+        | Orbs (Lightning, Frost, Dark, Plasma), Focus                   |
| **Watcher**  | Monk      | 75+        | Stances (Wrath 2x dmg, Calm energy, Divinity 3x), Scry, Retain |

## 🗺️ Map & Pathing System

- **Branching node map** — player chooses path each floor
- **Node types:** Normal enemies, Elite enemies, Rest sites, Shops, Events, Treasure, Boss
- **3 Acts + optional Act 4** — each with unique boss pool
- **Risk/Reward routing** — Elites are harder but drop relics

## 📈 Progression Systems

### In-Run

| Element     | Description                                             |
| ----------- | ------------------------------------------------------- |
| **Cards**   | Add/remove/upgrade via rewards, shops, events           |
| **Relics**  | 180+ passive bonuses collected throughout               |
| **Potions** | 3 slots; consumable combat bonuses                      |
| **Gold**    | Spend at shops for cards, relics, potions, card removal |

### Meta-Progression

- **Character unlocks** — Complete runs to unlock new characters
- **Ascension levels** — 0–20 increasing difficulty per character
- **Card/relic unlocks** — Score thresholds expand available pool
- **No permanent power gains** — pure skill progression

## 🌟 Key Innovations

1. **Enemy Intent System** — Enemies show their next action (attack, buff, debuff), enabling strategic planning
2. **Card Removal as Strategy** — Removing cards from deck is often more important than adding
3. **Branching Map Choice** — Players choose risk level by routing toward/away from elites
4. **Relic Synergies** — 180+ relics create emergent combos with card strategies
5. **Ascension Difficulty** — 20 levels of granular difficulty increase per character
6. **Popularized roguelike deckbuilding** — spawned entire subgenre (Balatro, Inscryption, Monster Train, etc.)
