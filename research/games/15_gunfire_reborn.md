# Gunfire Reborn (2021) — Duoyi Interactive

> **Genre:** FPS Roguelite | **Perspective:** First-person 3D | **Players:** 1–4 Online Co-op | **Steam Rating:** 93% Positive

## 🎮 Core Concept

A cell-shaded FPS roguelite with RPG elements. Choose a hero, grab randomized weapons, and clear procedural levels with up to 3 friends. Deep build crafting through scrolls, ascensions, weapon inscriptions, and hero talents.

### Inscription System

- Weapons drop with random **Inscriptions** (affixes/perks)
- Inscriptions can synergize with hero abilities
- Examples: "+30% damage to burning enemies", "10% chance to freeze on hit", "+1 projectile"
- **Gemini Inscription** — Rare inscription that fundamentally changes weapon behavior

### Elemental Fusion System (Key Innovation)

| Element 1     | + Element 2 | = Fusion Effect                       |
| ------------- | ----------- | ------------------------------------- |
| **Fire**      | + Corrosion | = **Manipulation** (increased damage) |
| **Fire**      | + Lightning | = **Explosion** (AoE burst)           |
| **Lightning** | + Corrosion | = **Miasma** (AoE DOT cloud)          |
| **Fire**      | + Ice       | = **Decay** (defense reduction)       |
| **Lightning** | + Ice       | = **Disintegration** (instant damage) |
| **Corrosion** | + Ice       | = **Catalysis** (chain reaction)      |

Combining two elements on the same enemy triggers fusion — rewards multi-element builds.

## 📈 Progression Systems

### In-Run

- **Scrolls** — Choose stat bonuses (3 per offering: Damage, Armor, HP + secondary effects)
- **Ascensions** — Major perks chosen between acts (character-specific ability upgrades)
- **Occult Scrolls** — Rare powerful modifiers (like relics in Slay the Spire)
- **Vault upgrades** — Weapon enhancement stations

### Meta-Progression: Talent Tree

| System                 | Currency     | Effect                         |
| ---------------------- | ------------ | ------------------------------ |
| **Talent Tree**        | Soul Essence | Permanent stat boosts per hero |
| **Account Level**      | XP           | Unlock talent tree tiers       |
| **Hero Mastery**       | Play time    | Unlock hero-specific abilities |
| **Weapon Proficiency** | Usage        | Unlock weapon variants         |

### Difficulty: Reincarnation Levels

- **8 Reincarnation levels** — Each adds modifiers (tougher enemies, new mechanics)
- Similar to Ascension in Slay the Spire or Boss Cells in Dead Cells

## 🌟 Key Innovations

1. **Elemental Fusion Combos** — Two elements combine for unique bonus effects
2. **FPS + Roguelite + RPG Hero Classes** — Each hero feels like a different game
3. **4-Player Co-op Build Synergy** — Teams can coordinate elemental combos across heroes
4. **Inscription Randomization** — Same weapon feels different each run
5. **Scroll System** — Simple but deep stat investment choices
6. **Cross-Platform** — PC, Console, Mobile with co-op
