# Void Bastards (2019) — Blue Manchu

> **Genre:** Strategy-Shooter Roguelite (FPS) | **Perspective:** First-person 3D | **Players:** Solo | **Steam Rating:** 78% Positive

## 🎮 Core Concept

A roguelite FPS set in the Sargasso Nebula. You play as expendable convicts ("clients") who must scavenge derelict spaceships to escape. Combines FTL-style nebula navigation with FPS exploration and crafting. Run-based with persistent crafting progress.

## 🗺️ Nebula Navigation — FTL-Style Map

- **Node-based map** — Choose which ship to board or skip
- **Ship types** visible with info: enemies, hazards, loot, layout
- **Resource consumption per jump** — food and fuel
- **Hazards** — Void whales, pirates, radiation zones
- Run out of food/fuel = death
- Creates constant tension between exploring and conserving

| **Colorblind** | Grayscale vision |

| **Weak** | Less HP |

## 📈 Progression — Persistent Crafting

### In-Run

- **Scavenge** — Collect food, fuel, ammo, crafting materials
- **Manage resources** — food/fuel deplete per jump
- **Choose fights** — Run from or fight enemies

### Meta-Progression (Persists Across Deaths)

| Persists                  | Lost on Death           |
| ------------------------- | ----------------------- |
| Crafted weapons/gadgets   | Food supply             |
| Crafting recipes unlocked | Fuel supply             |
| Merits (special currency) | Ammo                    |
| Story progress            | Current client's traits |

## 🎨 Art Style — Comic Book

- **Cell-shaded 3D** — Looks like a playable graphic novel

1. **FTL Map + FPS Exploration** — Strategic node-based travel + first-person ship scavenging
2. **Persistent Crafting** — Items you craft carry between deaths (clients)
3. **Ship System Interaction** — Hack power, security, doors on each ship
4. **Comic Book Art** — Unique visual identity
5. **Dark British Humor** — Tone sets it apart
6. **Strategic FPS** — Think first, shoot second
