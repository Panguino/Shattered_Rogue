# Rogue Legacy 2 (2022) — Cellar Door Games

> **Genre:** Platformer Roguelite | **Perspective:** 2D Side-scrolling | **Players:** Solo | **Steam Rating:** 93% Positive

## 🎮 Core Concept

You play as successive generations of a heroic lineage. Each heir has randomly generated traits, classes, and abilities. Die, spend gold on permanent castle upgrades, and send the next heir. Permanent progression is the core identity.

## 🏰 Castle Upgrade System (Meta-Progression)

### Castle Structure

Spend gold earned in runs to build/upgrade:
| Category | Examples | Effect |
|----------|----------|--------|
| **HP** | HP upgrades, Vitality | Permanent max HP increase |
| **Damage** | Strength, Intelligence | Permanent damage increase |
| **Armor** | Armor nodes | Permanent damage reduction |
| **Class Unlocks** | Barbarian, Mage, etc. | Unlock new starting classes |
| **Equipment** | Smithy, Enchantress | Access to gear and runes |
| **Ability** | Soul Shop, Mastery | New abilities and skills |

### Rune System

Runes grant traversal/combat abilities:

- **Dash** — i-frame dash
- **Double Jump** — Extra air jump
- **Vault** — Ledge grab
- **Air Dash** — Horizontal air movement
- **Bouncing** — Bounce off enemies/spikes

### Equipment

Found during runs, crafted at Smithy:

- **Sword variants** — Different damage/speed/range
- **Armor sets** — Stat modifications
- **Rune combinations** — Customize movement abilities

## 📈 Architect & Charon

### The Architect

- Pay gold to **lock the castle layout** from the previous run
- Costs 40% of all gold earned
- Useful if you found a good map or know the boss location

### Charon

- Tax collector at the castle entrance
- Takes ALL remaining gold before entering the castle
- Forces you to spend everything before each run

## 🌟 Key Innovations

1. **Generational Lineage** — Each heir is unique with random traits/class
2. **Permanent Castle Upgrades** — Pioneered the "roguelite" meta-progression concept
3. **Traits as Humor + Gameplay** — Some traits are funny, some change gameplay, some do both
4. **Architect System** — Lock previous map layout for a cost
5. **Charon Gold Tax** — Forces investment before each run (no hoarding)
6. **Class Mastery** — Leveling up classes unlocks permanent bonuses
