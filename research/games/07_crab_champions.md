## 🧬 Upgrade System — The Core Loop

### Three Upgrade Paths (Choose 1 per island)

After clearing each island, choose from 3 options:

1. **Weapon Mods** (89 total) — Modify how your weapon fires
2. **Ability Mods** (44 total) — Modify grenades and special abilities
3. **Perks** (108 total) — Passive stat/gameplay bonuses

### Upgrade Stacking

- Upgrades STACK throughout a run
- Multiple copies of the same mod = exponential effects
- Example: Stack "Extra Projectile" 5x = shotgun becomes wall of bullets
- Example: Stack "Bouncing" + "Piercing" = projectiles that never stop

### Elemental System

| Element       | Effect                                    |
| ------------- | ----------------------------------------- |
| **Acid**      | Weakens enemies (take more damage)        |
| **Fire**      | Damage over time (burn)                   |
| **Ice**       | Freezes/slows enemies                     |
| **Lightning** | Slows + chains to nearby enemies          |
| **Water**     | Removes/amplifies other elemental effects |

### Special Abilities

- **Black Hole** — Pulls enemies together
- **Grappling Hook** — Fast traversal + combat repositioning
- **Ice Blast** — AoE freeze
- **Laser Beam** — Continuous energy beam
- Standard grenade can be modded 44 ways

## 🗺️ Level Structure

```
Starting Island → Clear waves → Choose upgrade →
Navigate island chain → Boss island (every few islands) →
Boss drops key → Key unlocks content
```

- Open 3D island arenas with parkour elements
- Movement: jump, slide, wall-run, grapple
- Difficulty scales with island count

## 📈 Progression Systems

### In-Run

- Weapon mods, ability mods, perks (stackable)
- Weapon drops from enemies/chests
- No in-run currency — pure upgrade choices

### Meta-Progression — "No-Grind" Philosophy

| Unlock Type            | How         | Notes                              |
| ---------------------- | ----------- | ---------------------------------- |
| **Weapons**            | Boss keys   | New weapon types                   |
| **Cosmetic Skins**     | Boss keys   | Visual only                        |
| **Game Modes**         | Progression | New challenges                     |
| **NO permanent power** | —           | No damage/HP upgrades between runs |

> **Design Philosophy:** "Skill and build-crafting should determine success, not how many hours you've ground." This is deliberately anti-meta-progression.

## 🌟 Key Innovations

1. **No Meta-Power Progression** — Unlocks are content/cosmetic only. Radical design choice
2. **Deep Stacking System** — 89+44+108 = 241 upgrades that stack multiplicatively
3. **Fluid 3D Movement** — Parkour, sliding, wall-running, grappling in arenas
4. **Solo Dev Success** — Entire game made by one developer (Noisestorm)
5. **Elemental Interactions** — Water removes/amplifies other elements
6. **Co-op Scaling** — Difficulty balanced for 1–4 players automatically
