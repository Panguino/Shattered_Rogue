# Dead Cells (2018) — Motion Twin

> **Genre:** Metroidvania Roguelite | **Perspective:** 2D Side-scrolling | **Players:** Solo | **Steam Rating:** 95% Positive

## 🎮 Core Concept

A failed alchemist's soul inhabits headless bodies in an ever-changing castle. Fast-paced 2D combat with roguelite permadeath and Metroidvania-style permanent ability gates between runs.

## 🔫 Weapon System

### Weapon Categories

| Category         | Examples                                         | Slot              |
| ---------------- | ------------------------------------------------ | ----------------- |
| **Melee**        | Broadsword, Rapier, Twin Daggers, Shovel, Scythe | Main hand         |
| **Ranged**       | Bow, Throwing Knife, Fire Brands, Hokuto's Bow   | Main hand         |
| **Shields**      | Infantry Shield, Rampart, Cudgel                 | Main hand (parry) |
| **Skills/Traps** | Sinew Slicer, Wolf Trap, Turrets, Grenades       | Skill slots (2)   |

### Weapon Affixes

Every weapon rolls with random affixes:

- **"+100% damage to poisoned enemies"**
- **"Shoots an arrow on critical hit"**
- **"Spreads oil on the ground"**
- These create emergent synergy — poison weapon + "+100% to poisoned" = devastating

### Stat Scaling (Scrolls)

Weapons scale with one of three stats:

- **Brutality (Red)** — Melee damage, HP
- **Tactics (Purple)** — Ranged/trap damage, HP
- **Survival (Green)** — Shield/slow weapons, most HP

Scrolls found in levels let you invest in stats, encouraging build commitment.

## 📈 Progression Systems

### In-Run

- **Weapons & Skills** — Found in levels, shops, cursed chests
- **Scrolls** — Stat investment (Brutality/Tactics/Survival)
- **Mutations** — 3 slots for passive perks (chosen between biomes)
- **Gold** — Shops, doors, food

### Meta-Progression

| System                | Currency   | Effect                                                     |
| --------------------- | ---------- | ---------------------------------------------------------- |
| **Blueprint Unlocks** | Cells      | Permanently add weapons/mutations to item pool             |
| **Flask Upgrades**    | Cells      | More healing charges between biomes                        |
| **Gold Reserves**     | Cells      | Keep gold percentage between runs                          |
| **Forge**             | Cells      | Improve starting weapon quality (+ to ++)                  |
| **Runes**             | Boss kills | Permanent traversal abilities (vine climb, teleport, etc.) |

### Boss Cells (Difficulty)

- 0BC → 5BC difficulty levels
- Each adds: tougher enemies, fewer healing, new biomes, new bosses
- 5BC is the "true" ending difficulty

## 💡 Key Combat Mechanics

- **Rally System** — After taking damage, quickly deal damage to recover lost HP (orange health)
- **Parry** — Shields can parry for massive counter-damage and stun
- **Dodge Roll** — i-frames, but limited
- **Speed Bonus Doors** — Timed doors reward fast play with bonus items/cells
- **Cursed Chests** — Free item, but next 10 kills = instant death if hit

## 🌟 Key Innovations

1. **Rally/Recovery System** — Regain recently lost HP by attacking aggressively
2. **Metroidvania Gates** — Permanent rune abilities (vine, teleport, ram) open new paths in future runs
3. **Stat-Affixed Weapons** — Random affixes create emergent build synergies
4. **Score Scaling (Brutality/Tactics/Survival)** — Choose a build path each run
5. **Speed Incentives** — Timed doors reward aggressive play without forcing it
6. **"No Sequel, Just Updates"** — 5+ years of free + DLC content post-launch
