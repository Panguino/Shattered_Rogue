# Nova Drift (2024) — Chimeric

> **Genre:** Arcade Roguelite Space Shooter | **Perspective:** Top-down 2D | **Players:** Solo | **Steam Rating:** 95% Positive

## 🎮 Core Concept

A modern _Asteroids_ with roguelite build-crafting depth. Pilot a bio-mechanical ship through arena waves of enemies, leveling up to select Gear (Weapon + Shield + Body) and Mods that define your playstyle. Deep modular build system.

## 🧬 Mod System (100+)

After core Gear, level-ups offer Mods from a semi-randomized pool:

- **Homing** — Projectiles track enemies
- **Piercing** — Projectiles pass through enemies
- **Bouncing** — Projectiles ricochet
- **Split** — Projectiles split on impact
- **Siege** — Increased damage, reduced fire rate
- **Amplify** — Shield effects are stronger
- **Corrosion** — Attacks reduce enemy defense
- Many mods have **trade-offs** (pros and cons)

### Build Examples

- **Drone Commander:** Carrier body + Drone weapon + Drone-enhancing mods
- **Ram Tank:** Hullbreaker body + Shockwave shield + Collision mods
- **Sniper:** Rail Gun + Bastion shield + Siege + Focus mods
- **Melee Berserker:** Swords + Halo shield + Firefly body

## 📈 Progression

### In-Run

- XP orbs from kills → Level up → Choose Gear or Mod
- First 3 level-ups: Weapon, Shield, Body selection
- Subsequent: Mods from randomized pool

### Meta-Progression

| System                   | How              | Purpose                             |
| ------------------------ | ---------------- | ----------------------------------- |
| **Score Ranking**        | Score per run    | Unlock new Weapons, Shields, Bodies |
| **Content Unlocks**      | Score thresholds | Expand available Gear and Mods      |
| **Difficulty Selection** | Progression      | Higher challenge modes              |
| **No permanent power**   | —                | Score/skill-based only              |
