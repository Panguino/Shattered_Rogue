# Noita (2020) — Nolla Games

> **Genre:** Physics Roguelike | **Perspective:** 2D Side-scrolling | **Players:** Solo (Modded MP) | **Steam Rating:** 93% Positive

## 🎮 Core Concept

Every pixel in the world is physically simulated. Dig, explode, burn, and melt your way through procedurally generated caves using custom-crafted wands. Knowledge is the ultimate form of progression.

## 🪄 Wand Crafting System — The Core Innovation

### Wand Properties

Every wand has base stats:
| Stat | Effect |
|------|--------|
| **Shuffle** | Yes/No — spells cast in order or random |
| **Spells/Cast** | How many spells fire per click |
| **Cast Delay** | Time between spell groups |
| **Recharge Time** | Time to refill all spells |
| **Mana** | Resource for casting |
| **Mana Charge** | Mana regen rate |
| **Spread** | Accuracy deviation |
| **Capacity** | Spell slots available |

### Spell Types (400+)

| Category       | Examples                                           |
| -------------- | -------------------------------------------------- |
| **Projectile** | Spark Bolt, Magic Arrow, Bubble Spark, Death Cross |
| **Static**     | Circle of Vigor, Tentacle, Earthquake              |
| **Modifier**   | Homing, Bouncing, Piercing, Heavy Shot, Light Shot |
| **Multicast**  | Double Spell, Triple Spell, Formation (Y, T, star) |
| **Material**   | Water, Blood, Oil, Acid, Lava, Toxic Sludge        |
| **Timer**      | Delay trigger, Timer trigger, Proximity Mine       |
| **Special**    | Divide By, Alpha, Omega, Gamma, Mu                 |
| **Utility**    | Teleport, Shield, Light, Nuke                      |

### Wand Crafting Examples

```
[Spark Bolt] + [Homing] + [Heavy Shot] = Homing heavy spark bolt
[Triple Spell] + [Spark Bolt] + [Spark Bolt] + [Spark Bolt] = Triple rapid fire
[Nuke] + [Divide By 10] = 10 simultaneous nukes (WARNING: will kill you)
[Chainsaw] + [Chainsaw] + [Chainsaw] = Reduces cast delay to near-zero
```

### "Infinite Spells" Wand Craft

```
[Always Cast: Chainsaw] → Slot: [Spark Bolt + Homing]
= Machine gun of homing bolts (chainsaw removes delay)
```

## ⚗️ Material / Physics System

### Pixel Simulation

- Every pixel is one of 50+ materials
- Materials interact: oil + fire = burning, water + toxic = diluted, acid dissolves everything
- Liquids flow, gases rise, fire spreads, ice melts
- Terrain can be destroyed, created, transformed

### Key Materials

| Material          | Behavior                                       |
| ----------------- | ---------------------------------------------- |
| **Water**         | Flows, extinguishes fire, conducts electricity |
| **Oil**           | Flows, extremely flammable                     |
| **Acid**          | Dissolves terrain and enemies                  |
| **Lava**          | Burns everything, creates rock when cooled     |
| **Toxic Sludge**  | Poisons, stains player                         |
| **Blood**         | Cosmetic liquid from enemies                   |
| **Polymorphine**  | Transforms things it contacts                  |
| **Ambrosia**      | Protective coating (rare)                      |
| **Teleportatium** | Teleports things it contacts                   |

## 🦸 Perk System (Choose 1 per Holy Mountain)

After each biome, visit the Holy Mountain to:

- Edit wands freely
- Buy spells/wands
- Choose 1 perk from 3 options

### Perk Examples

- **Exploding Corpses** — Dead enemies explode
- **Glass Cannon** — 5x damage dealt/received
- **Unlimited Spells** — Spells never run out (but still cost mana)
- **Edit Wands Everywhere** — Craft wands outside Holy Mountain
- **Extra Life** — Revive once
- **Freeze Field** — Freezes nearby projectiles
- **Vampirism** — Blood heals you, no heart pickups
